# Joypad Build Guides

DIY guides for building your own Joypad adapters.

## Adapter Guides

| Guide | Description | Difficulty |
|-------|-------------|------------|
| [USB2GC](usb2gc.md) | USB controller → GameCube | ⭐ Easy |
| [USB2USB](usb2usb.md) | USB controller → USB (SInput) | ⭐ Easy |
| [BT2USB](bt2usb.md) | Bluetooth controller → USB | ⭐⭐ Medium |
| [N642DC](n642dc.md) | N64 controller → Dreamcast | ⭐⭐ Medium |
| [DC2USB](dc2usb.md) | Dreamcast controller → USB | ⭐⭐ Medium |
| [GC2USB](gc2usb.md) | GameCube controller → USB | ⭐⭐ Medium |
| [N642USB](n642usb.md) | N64 controller → USB | ⭐⭐ Medium |

## Guide Format

Each guide includes:
- **Overview** — What you're building
- **Parts List** — Everything you need (with links)
- **Wiring Diagram** — Visual connection guide
- **Assembly Steps** — Step-by-step instructions
- **Firmware Flash** — How to install Joypad OS
- **Testing** — Verify it works
