# Joypad Build Guides

DIY guides for building your own Joypad adapters.

## Adapter Guides

### USB Output (play on PC/modern devices)
- [x] [USB2USB](usb2usb.md) — USB controller → USB (SInput) ⭐ Easy
- [x] [BT2USB](bt2usb.md) — Bluetooth controller → USB ⭐ Easy
- [ ] DC2USB — Dreamcast controller → USB ⭐⭐ Medium
- [x] [GC2USB](gc2usb.md) — GameCube controller → USB ⭐⭐ Medium
- [x] [N642USB](n642usb.md) — N64 controller → USB ⭐⭐ Medium
- [x] [SNES2USB](snes2usb.md) — SNES controller → USB ⭐⭐ Medium

## Guide Format

Each guide includes:
- **Overview** — What you're building
- **Parts List** — Everything you need (with links)
- **Wiring Diagram** — Visual connection guide
- **Assembly Steps** — Step-by-step instructions
- **Firmware Flash** — How to install Joypad OS
- **Testing** — Verify it works
